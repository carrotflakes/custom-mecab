# custom-mecab



## Usage

Make sure `*ipadic-path*` and `*mecab-dict-index-path*` in `src/config.lisp`.

```
npm i
npm run clackup
```

## Installation

Install MeCab and ipadic.

```
ros install clack
ros install carrotflakes/cl-mecab
```
